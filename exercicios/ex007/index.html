Principais Formatações

vendo o que aprendi em partes

Negrito / destaque

Aqui para deixar o texto em negrito posso usar a tag B que significa bold eu acho tlg (não semântica)

Agora, para dar destaque a um termo, eu uso a tag STRONG (semântica)

Na prática no sentido visual, não muda absolutamente nada, mas para o HTML é muito importânte dar semântica as tags e textos (estilização fica com as CSS)

Itálico / ênfase

Aqui temos um termo em itálico, e posso usar a tag I para representar (não semântica)

Aqui temos um termo em ênfase, posso usar a tag EM para o representar (semântica)

Marcação de texto

Bem, aqui eu posso marcar um texto, com um marcador usando a tag MARK

Também posso usar a tag MARK com formas de cores diferentes usando as CSS

Textos grandes e pequenos

Bem, aqui eu posso usar a tag BIG para representar uma letra maior, mas ela é inviavel, já que ela vai se tornar obsoleta por não ter um sentido semântico claro (sendo uma forma que fica com as CSS)

Entretanto, por algum motivo a tag small sobrevive e segue firme e forte até hoje, o seu sentido semântico pode ser atribuido a letras miudas de contratos

Texto deletado / Inserido

Posso "riscar um texto" usando a tag DEL, tem sentido de texto que deve ser lido, mas não deve ser considerado na informação (semântica)

Também posso marcar como inserido usando a tag INS, que tem um sentido de algo que deve ser realmente lido com atenção (semântica)

Também temos a tag U, que deveria ter o mesmo sentido da tag INS, mas como ela não tem fim semântico e sim de forma, logo logo vai se tornar obsoleta (não semântica)

Texto sobrescrito / subescrito

Aqui posso colocar um texto sobrescrito com próposito de mexer com coisas do tipo X20+3

Aqui posso usar um texto subescrito com a mesma finalidade, como por exemplo H2O
