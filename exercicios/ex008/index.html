Tipos de formatações de texto parte dois

Segunda parte de seus tipos e exemplificações

Código / pré formatação

Isso é utíl em casos em que se deve representar algum código de programação, como por exemplo:

    
            #include <stdio.h>
            int main(void)
            {
               printf("Hello World!\n");
               return 0;
            }

Citações

Aqui existe dois tipos de citações, a que se pode usar a tag Q (do inglês "quote")
geralmente usada em textos, e a blockquote (uma citação também, só que mais completa) como por exemplo:

BASIC é uma linguagem imperativa de alto nível, pertencente à terceira geração, originalmente compilada (apesar de suas implementações em microcomputadores ter disseminado a versão interpretada) e não estruturada, por ter sido fortemente baseada em Fortran II.

Abreviação

Bem, aqui posso abreviar siglas e colocar elas por extenso como essa aqui, usando a tag "abbr":

RNP é uma rede brasileira que conecta instituições de ensino, pesquisa e inovação com internet e serviços seguros.

Texto invertido

Tem quem diz que essa praga é inutil, mas depois de uma breve pesquisa, essa tag que inverte a ordem de seus caracteres, pode ser usada por exemplo, com idiomas que começam da direita para a esquerda, usando a tag BDO com o parametro right-to-left ou left-to-right, resumindo util a suporte de idiomas
